# Projeto Integrador – Sistema de Agendamento

Este repositório contém os arquivos do Projeto Integrador do curso de Tecnologia em Análise e Desenvolvimento de Sistemas.

## Descrição

O projeto consiste no desenvolvimento de um sistema web responsivo para agendamento de serviços, permitindo cadastro de usuários, controle de horários e gerenciamento de agendamentos.

A solução será desenvolvida utilizando arquitetura cliente-servidor, com backend baseado em webservice REST, frontend web responsivo e banco de dados para persistência das informações.

## Objetivo

Aplicar os conceitos das disciplinas do semestre, incluindo:

- Arquitetura de software
- Gerenciamento de projetos
- Linguagens de servidor
- Sistemas distribuídos
- Web standards

## Estrutura do repositório

/backend → código do servidor  
/frontend → interface web  
/docs → documentação do projeto
